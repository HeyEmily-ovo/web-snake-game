# 贪吃蛇小游戏

一个使用原生 HTML、CSS 和 JavaScript 实现的简单 Web 端贪吃蛇小游戏，无需安装依赖，直接在浏览器中打开即可运行。

## 功能特性

- 原生静态页面实现，无需构建工具
- 支持键盘方向键和 `W/A/S/D` 控制
- 支持移动端触控方向按钮
- 吃到食物后增长并增加分数
- 支持穿过边缘从另一侧出现
- 撞到自己后游戏结束
- 记录浏览器本地个人最高分

## 文件结构

```text
snaketest/
├── index.html   # 页面结构
├── style.css    # 页面样式与移动端布局
├── script.js    # 游戏逻辑与交互控制
└── README.md    # 项目说明
```

## 运行方式

### 方式一：直接打开

直接用浏览器打开 `index.html` 即可运行。

### 方式二：本地静态服务

如果你想通过本地服务访问，也可以在当前目录启动一个简单静态服务，例如：

```bash
python -m http.server 8000
```

然后在浏览器访问：

```text
http://localhost:8000
```

## 操作说明

### 电脑端

- `↑` `↓` `←` `→`
- `W` `A` `S` `D`

### 移动端

- 点击页面中的方向按钮 `↑ ← ↓ →`

## 游戏规则

- 点击“开始游戏”开始
- 吃到食物会增加分数并让蛇变长
- 蛇碰到边缘会从对侧穿出
- 撞到自己会结束游戏
- 游戏过程中不能直接反向掉头

## 最高分说明

游戏会使用浏览器的 `localStorage` 保存当前设备和当前浏览器下的个人最高分。

如果你清除浏览器本地存储，最高分记录也会一并清除。
